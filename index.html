Dashboard Kronobergs län

Livehämtade nyckeltal för länets kommuner med jämförelser mot Kronobergs län och riket.

Vald huvudkommun: Växjö
Klar att ladda

Källa: Kolada API v3 samt Havs- och vattenmyndighetens API för badplatser och badvatten. Kronobergs län hämtas som Kolada-värde där det finns, annars beräknas det live från länets kommuner. Riket hämtas som livevärde från Kolada.
